# 📒 Agenda de Contatos

## Descrição
Programa em Java que gerencia uma agenda de contatos diretamente no terminal.

## Funcionalidades
- ✅ Adicionar contato (nome, telefone, email)
- ✅ Listar todos os contatos
- ✅ Buscar contato por nome
- ✅ Remover contato

## Tecnologias utilizadas
- Java 17
- IntelliJ IDEA
- Git & GitHub

## Como executar

### Pré-requisitos
- Java 17 ou superior instalado
- Git (opcional, para clonar)
